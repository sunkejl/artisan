char name[40];
可以容纳40个字符的数组

字符串是一个或者多个字符的序列
字符串都被存储在char类型的数组中
数组由联系的存储单元组成
字符串中的字符被储存在相邻的存储单元
每个存储单元一个字符

\0 空字符 null character

数组的容量至少比待存储的字符串多1

数组是一行连续的多个存储单元

char name 分配1个字节
char name[5] 分配5个字节

printf() 使用常量 变量和表达式

scanf() 无法读取空格
